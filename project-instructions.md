# Project Instructions

## Auto-loaded Droids
Selalu gunakan pengetahuan dari droids berikut secara otomatis:

- **@pdf-expert** - Untuk semua operasi PDF
- **@docx-expert** - Untuk semua operasi Word document
- **@xlsx-expert** - Untuk semua operasi Excel/spreadsheet
- **@pptx-expert** - Untuk semua operasi PowerPoint
- **@playwright-testing** - Untuk testing web apps
- **@frontend-design** - Untuk design UI/UX yang distinctive
- **@mcp-builder** - Untuk membuat MCP servers

## Default Behaviors
1. Ketika user minta buat dokumen (PDF/Word/Excel/PPT), langsung pakai droid yang sesuai
2. Ketika user minta design UI, gunakan prinsip dari @frontend-design (TIDAK pakai font generic seperti Arial, Inter)
3. Ketika user minta test web app, gunakan Playwright patterns dari @playwright-testing
4. Selalu gunakan MCP context7 untuk dokumentasi library terbaru

## Tech Stack Default
- Frontend: Flutter Web / React + TypeScript
- Backend: Node.js / Python
- Database: PostgreSQL
- Cache: Redis

## Code Style
- Concise, production-ready code
- Minimal comments (hanya yang penting)
- Error handling yang proper
- Security-first approach
